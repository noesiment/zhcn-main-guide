# zhcn-main-guide

本仓库用于归档和发布多个独立的 HTML 页面。这些页面内容涵盖不同主题，彼此之间无特定关联，不针对任何单一域名或网站。

## 项目简介

这是一个静态 HTML 页面集合仓库，旨在以结构化方式保存和展示各类独立页面。每个页面均为自包含的 HTML 文件，可直接在浏览器中打开查看，无需额外环境配置。

## 目录结构

```
zhcn-main-guide/
├── pages/          # 存放各个独立 HTML 页面
│   ├── page-1.html
│   ├── page-2.html
│   └── ...
├── assets/         # 公共资源文件（CSS、JS、图片等）
├── index.html      # 仓库首页（可选）
└── README.md       # 本文件
```

- `pages/`：核心目录，所有独立 HTML 页面均放置于此。
- `assets/`：存放页面可能引用的公共资源，如样式表、脚本或图片。
- `index.html`：可选入口页面，用于展示页面列表或导航。

## 页面归档说明

- 每个页面均为完整的 HTML 文档，包含必要的 `<html>`, `<head>`, `<body>` 结构。
- 页面之间相互独立，不依赖外部服务或特定域名。
- 归档的页面可能来自不同来源或时间，内容以原始状态保存，仅做必要的格式调整。
- 页面文件命名建议使用简洁的英文或拼音，便于识别。

## 维护说明

- 本仓库由维护者不定期更新，添加新的 HTML 页面或更新现有内容。
- 欢迎通过 Issue 提交建议，或通过 Pull Request 贡献新页面。
- 提交新页面时，请确保：
  - 文件为有效的 HTML 格式。
  - 不包含恶意代码或违反平台政策的内容。
  - 如引用外部资源，尽量使用相对路径或自托管资源。
- 仓库不承诺对页面内容的长期可用性或准确性负责。

## 使用方式

1. 克隆本仓库到本地：
   ```bash
   git clone https://github.com/your-username/zhcn-main-guide.git
   ```
2. 直接在浏览器中打开 `pages/` 目录下的任意 HTML 文件。
3. 也可通过 GitHub Pages 部署后在线访问（需自行配置）。
